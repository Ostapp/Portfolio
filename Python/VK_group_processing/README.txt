С помощью данного скрипта можно скачать посты в нужный вам отрезок времени из любой открытой группы Вконтакте, а также получить статистику встречаемости данных ключевых слов по интересующим вас датам. Данная статистика представляется в виде графика. Скрипт также анализирует данные посты и по другим параметрам. Для более подробной информации см. файл под названием  "Техническое-задание". 

Скрипт находится в файле vk_processor.py

Скрипт функционален, выполняет свою работу, однако я ещё не подготовил его для удобного пользования другими юзерами.
